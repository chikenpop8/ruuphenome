What is FDR?
False Discovery Rate — the % of reported compounds we're okay with being wrong.

How it works here:
We test real compounds against fake "decoy" compounds that shouldn't appear in any real sample. If a decoy sneaks through, it's a false positive. FDR controls how many of those we allow.

Why it matters:
Set it too high → noise gets reported as real metabolites → your results look impressive but aren't trustworthy.
Set it too low → only the most obvious metabolites pass → safer, but you might miss real signals.

5% is the standard used in proteomics and genomics worldwide.
